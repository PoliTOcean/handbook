## 🛠️ Local preview

This handbook is built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/).
To preview it locally:

```bash
pip install mkdocs-material
mkdocs serve
# open http://127.0.0.1:8000
```

The site is built and deployed automatically to GitHub Pages on every push to
`main`.
